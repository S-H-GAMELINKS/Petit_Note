# PetitNote
- 1スレッド1ログファイル形式のスレッド式の画像掲示板です。  
- PaintBBS NEOとChickenPaintが使えるお絵かき掲示板です。
- こんなコードですが、著作権は放棄しません。  
- ウェブに設置して使用しないでください。ログが消失したり、サーバに負荷がかかってもなにもできません。  
- 動作確認はローカルサーバで。
